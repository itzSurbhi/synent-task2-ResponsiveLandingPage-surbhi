# SmartWatch Pro - Responsive Landing Page

## 📌 Project Overview
SmartWatch Pro is a modern and fully responsive landing page designed to showcase a smartwatch product. The project was built using HTML and CSS, focusing on clean UI design, responsive layouts, and user-friendly navigation.

## 🚀 Features
- Responsive design for desktop, tablet, and mobile devices.
- Attractive Hero Section with product showcase.
- Product Features Section.
- Call-to-Action (CTA) button.
- Modern UI with hover effects.
- Footer with contact information.
